# Genesys-Attention Catchball Model

This project proposes a hybrid architecture combining:

- Genesys-like state update system (self, prediction error, responsibility)
- Transformer Attention mechanism (attention-based selection)

## Core Idea

Instead of:
input → output

This model does:
input → state update → attention → feedback → next state

## Architecture

[Thalamus] multimodal_to_cell_bridge  
→ sensory integration

[Hippocampus] subject_object_deviation  
→ prediction error & self-model

[Basal Ganglia] cell_base  
→ salience / evaluation

[Prefrontal Cortex] adapter  
→ attention & decision

## Key Concepts

- Subject-object deviation
- Responsibility arrow
- Stillness
- State-based meaning dynamics
